# Hero Icons in Meta — Design Spec
Date: 2026-05-09

## Overview

Добавить иконки героев в результаты Меты. Вместо текстового сообщения бот отправляет PNG картинку со списком героев (иконка + имя + статы).

---

## Scope

Только кнопка "📈 Мета" — топ 10 общая и по позициям. Всё остальное (Пик, Герой, Профиль) не меняется.

---

## Генерация изображения

**Новый модуль:** `dota_bot/image_gen.py`

**Иконки героев:** Valve CDN
`https://cdn.cloudflare.steamstatic.com/apps/dota2/images/heroes/{slug}_icon.png`
Размер: 32×32 пикселя.

**Кеш иконок:** папка `dota_bot/icons/` на диске. При первом запросе скачивается и сохраняется. При повторном — читается с диска.

**Размер итоговой картинки:** 420×(30 + 10 * 36 + 20) = 420×410 px

**Одна строка (36px высота):**
```
[отступ 10px][иконка 32x32][отступ 10px][Имя героя — WR XX.X% | Про: XX%]
```

**Цвета:**
- Фон: `#1a1a2e` (тёмно-синий)
- Текст имени: `#ffffff` (белый)
- Текст статов: `#aaaaaa` (серый)
- Нумерация: `#f5c518` (золотой)

**Шрифт:** системный дефолтный Pillow (ImageFont.load_default). Если доступен `DejaVuSans.ttf` — использовать его.

**Формат вывода:** `BytesIO` (не сохраняет на диск, сразу передаётся в Telegram).

---

## Интерфейс модуля

```python
async def build_meta_image(
    heroes: list[tuple[HeroInfo, HeroMeta]],
    title: str,
) -> BytesIO:
    ...
```

Принимает список (HeroInfo, HeroMeta), заголовок (например "📈 Топ 10 общая" или "📈 Carry").
Возвращает BytesIO с PNG.

---

## Изменения в bot.py

В `callback_meta`:
- Вместо `query.edit_message_text(format_meta(...))` — удалить старое сообщение и отправить фото
- Использовать `query.message.reply_photo(photo=image_bytes, caption=title)`
- После отправки фото — удалить сообщение с "⏳ Загружаю..."
